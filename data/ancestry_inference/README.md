## README
This directory includes source data for Supplementary Figures 2a (ICGC), Supplementary Figure 2b (METABRIC) and Supplementary Figure 3c (Hartwig). Source data includes principal components 1 and 2 from principal component analysis of genotypes of 128 ancestry informative makers with reference populations provided by Philips et al.

### Columns
**sample:** sample name\
**PC1:** principal component 1\
**PC2:** principal component 2\
**ancestry:** ancestry population assigned in reference population or cohort name for samples of interest without known ancestry
